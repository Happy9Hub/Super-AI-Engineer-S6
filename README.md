# 🤖 Super AI Engineer Season 6 - Portfolio

Welcome to my project repository for **Super AI Engineer Season 6**. This repository contains the machine learning pipelines, models, and data analysis scripts I developed during the hackathon challenges, successfully advancing to the Level 2 interview stage. 

As a Senior Software Engineer with a decade of experience, my focus in these projects was not only on model accuracy but also on writing clean code and building scalable pipelines.

## 🚀 Featured Projects

### 👁️ Computer Vision
* **House Recognition:** High-accuracy image classification pipeline leveraging EfficientNetV2, Test-Time Augmentation (TTA), and Pseudo-Labeling.
* **2026 MP Election OCR:** Document digitization and optical character recognition pipeline for extracting text from structured election documents.

### 🗣️ Natural Language Processing & GenAI
* **Thai Word Segmentation:** End-to-end word segmentation pipeline transitioning from classical CRF (with N-gram feature extraction) to Deep Learning (BiLSTM) and state-of-the-art PyThaiNLP integration.
* **FahMai RAG Challenge:** Implemented a Retrieval-Augmented Generation (RAG) system for intelligent document retrieval and question answering.

### 📊 Data Science & Predictive Modeling
* **Heart Disease Prediction:** Predictive modeling and classification based on tabular health data.
* **Housing Income Analysis:** Exploratory Data Analysis (EDA) and trend prediction for housing income datasets.

## 🛠️ Tech Stack & Tools
* **Languages:** Python
* **Machine Learning:** TensorFlow, Keras, Scikit-learn, PyThaiNLP, sklearn-crfsuite
* **Data Processing:** Pandas, NumPy
* **Infrastructure/Ops (Background):** Docker, Node.js, SQL
